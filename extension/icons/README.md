# Extension Icons

This directory needs three icon files:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Quick Setup

### Option 1: Use the script (requires ImageMagick)
```bash
../create-icons.sh
```

### Option 2: Create manually
Use any image editor to create three PNG files with the sizes above. A simple colored square with "NR" text works fine for development.

### Option 3: Use an online tool
1. Visit https://www.favicon-generator.org/
2. Upload any image or create a simple design
3. Download the generated icons
4. Rename and place them in this directory
